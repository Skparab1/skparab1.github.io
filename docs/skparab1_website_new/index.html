Snake Game

A serverless, graphics-based game of snake with a leaderboard


Snake game brings on the next generation of graphics based web games. Built with a HTML body and canvas display controlled by JS, Snake game boasts a sharp, vivid, high quality display powering an entertaining game. In addition, snake features a live ranked database, allowing players to see their scores comparatively.

Developed by Shubham Parab, Anindit Pati, Agastya Pawate, Advaita Guruprasad, and Anish Lakkapragada

Forum message sender

A serverless forum built with a github database


Skparab1 forum embodies messages, replies, colors, and reply threads all contained in a website built with a JS and HTML frontend with a Github actions backend. While innovatively using a serverless backend with a Github database, Skparab1 forum is an open, usable communication space. Skparab1 forum also has a modern ui, with square/circle, and light/dark themes.

Developed by Shubham Parab and Anish Lakkapragada

About me

Hello there! I'm Shubham Parab, a 15 year old in San Jose, California. I am an aspiring web developer and general programmer who mainly uses Python, HTML, JavaScript, and CSS, but also knows Java and Pascal.

I like making special effects for websites, and using canvas to generate graphics. I also like making games and other interactive, graphics based software mainly in javascript and HTML.

I also enjoy mountain biking, hiking, and taking photos, in addition to programming
